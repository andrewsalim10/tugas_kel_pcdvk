# **Proyek Akhir**

Proyek akhir berdasarkan topik segmentasi citra, klasifikasi citra (MobileNetV3-Small), *object detection* dan *counting* (Mask R-CNN). Tujuan proyek akhir adalah mengenali dan menghitung penyakit pada daun apel. Terdapat pada *folder* proyek_akhir. Demonstrasi dilakukan dengan menjalankan kode _demo_model.py
